# Tilt UGC v2 — Setup Guide

## 1. Run SQL Migrations

Go to Supabase → SQL Editor → paste and run everything in `supabase-migrations-v2.sql`.

---

## 2. Google Sheet Template

Create a tab called **Briefs** in your Google Sheet with these exact column headers in Row 1:

| A | B | C | D | E | F | G | H | I |
|---|---|---|---|---|---|---|---|---|
| Video ID | Creator | Delivery | Hook | Script | CTA | Due date | Assigned date | Status |

Column details:
- **Video ID**: Manual. Use format like `YZ-001`, `KF-002`, `NY-003`
- **Creator**: Exactly matching a name in your creators table (e.g. `Yzzy`, `Kofi`, `Neya`)
- **Delivery**: e.g. `Talking head`, `Green screen`, `Overlaid text`
- **Hook**: The opening line (first 3 seconds)
- **Script**: Full body copy
- **CTA**: Call to action text
- **Due date**: Format: `YYYY-MM-DD` (e.g. `2026-04-10`)
- **Assigned date**: Auto-fill with `=TODAY()` or type the date
- **Status**: Leave blank — Make.com will set it to `assigned`, app updates to `posted`

- Row number: From Step 4 result

## 5. EmailJS Invoice Template

### Create a new template in EmailJS (separate from the idea notification template)

Template name: `Tilt Invoice`

**To email:** `{{to_email}}`

**Subject:** `Invoice from {{creator_name}} — {{invoice_date}}`

**Body:**
```
INVOICE

From: {{creator_name}}
Date: {{invoice_date}}
Period: {{invoice_period}}

─────────────────────────────

LINE ITEMS:

{{line_items}}

─────────────────────────────

TOTAL: {{total}}

─────────────────────────────

PAYMENT DETAILS:
{{bank_details}}

─────────────────────────────

This invoice was automatically generated by Tilt UGC.
```

Copy the **Template ID** and paste it into `src/App.jsx` as `EMAILJS_INVOICE_TEMPLATE_ID`.

## 6. Update App.jsx Config

Fill in these constants at the top of `src/App.jsx`:

```js

const EMAILJS_TEMPLATE_ID = "template_xxx";         // Idea notification template
const EMAILJS_INVOICE_TEMPLATE_ID = "template_yyy"; // Invoice template (new)

```

## 7. Set Creator Rates

In the admin app, go to each creator's profile → Edit profile → set:
- **Rate per video (£)**: e.g. `25`
- **Payment details**: Their bank name, sort code, account number, or PayPal

---

## 8. Deploy

```bash
git add src/App.jsx supabase-migrations-v2.sql SETUP-V2.md
git commit -m "v2: remove briefs, add invoice, bonus, sheets integration, preview mode"
git push origin main
```

Vercel auto-deploys in ~60s.
